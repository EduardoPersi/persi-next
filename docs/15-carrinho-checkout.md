# 15 — Carrinho e Checkout

## Objetivo
Criar um fluxo de compra rápido, confiável e com alta conversão.

## Mini Carrinho
Deve exibir:
- produtos
- quantidade
- subtotal
- remover item
- botão para carrinho
- botão finalizar compra

## Carrinho

Funcionalidades:
- alterar quantidade
- remover itens
- aplicar cupom
- calcular frete
- atualizar automaticamente

## Checkout

Prioridades:
1. Login opcional
2. Endereço
3. Frete
4. Pagamento
5. Revisão
6. Confirmação

## Pagamentos

Suportar:
- PIX
- Cartão
- Boleto (quando disponível)

## Segurança

- validar preços no servidor
- validar estoque
- impedir manipulação do carrinho

## UX

- poucos campos
- mensagens claras
- carregamento rápido
- sem recarregar a página quando possível

## Checklist

- responsivo
- acessível
- seguro
- rápido
