# 04 — SEO

## Objetivo

Garantir que o novo front-end preserve e melhore o SEO da Persi Materiais.

## Regras Gerais

Toda página deve possuir:

- Title único.
- Meta Description.
- H1 único.
- URL amigável.
- Canonical.
- Open Graph.
- Robots corretos.

## Produtos

Cada produto deve possuir:

- Schema Product.
- Offer.
- Brand.
- SKU.
- Disponibilidade.
- Preço.

## Categorias

As categorias devem conter:

- Introdução.
- Texto útil.
- Links internos.
- Breadcrumb.
- Produtos.

## Marcas

As páginas de marcas devem ter:

- Logo.
- Descrição.
- Produtos relacionados.
- Conteúdo exclusivo.

## SEO Local

Priorizar cidades atendidas:

- Jundiaí
- Itupeva
- Louveira
- Cabreúva
- Itatiba
- Várzea Paulista

Evitar conteúdo duplicado.

## Sitemap

Gerar automaticamente.

## Robots

Permitir indexação apenas do conteúdo relevante.

## Performance

SEO depende também de:

- LCP
- CLS
- INP
- TTFB

## Imagens

Todas as imagens devem possuir:

- alt descritivo;
- nome organizado;
- dimensões corretas;
- formato otimizado.

## Links

Priorizar links internos entre:

- categorias;
- produtos;
- marcas;
- conteúdo institucional.

## Migração

Antes da publicação:

- validar redirects;
- comparar URLs;
- testar Search Console;
- revisar sitemap;
- revisar schema.
