# 14 — Componentes de Produto

## Objetivo

Definir o padrão para todos os componentes relacionados aos produtos.

## Componentes

- ProductCard
- ProductGallery
- ProductPrice
- ProductBadge
- ProductVariation
- ProductStock
- ProductDescription
- ProductSpecifications
- RelatedProducts
- UpsellProducts
- CrossSellProducts

## ProductCard

Exibir:

- imagem;
- nome;
- preço;
- marca.

Na Home:

- não exibir botão de compra.

## Página de Produto

Prioridade visual:

1. Galeria
2. Nome
3. Preço
4. Pix
5. Parcelamento
6. Estoque
7. Comprar
8. Frete

## Variações

Permitir:

- tamanho;
- cor;
- modelo;
- outras variações do WooCommerce.

## Produto sem estoque

Substituir botão principal por contato via WhatsApp.

## Compre Junto

Deve permitir:

- seleção independente;
- quantidade individual;
- cálculo automático;
- um único botão de compra.

## Galeria

- zoom;
- troca rápida;
- miniaturas;
- imagens otimizadas.

## Checklist

- responsivo;
- acessível;
- rápido;
- reutilizável;
- tipado.
